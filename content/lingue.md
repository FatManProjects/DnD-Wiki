Le lingue parlate all'interno del Totus sono le classiche definite per razza, quindi:

| Razza    | Lingua   |
| -------- | -------- |
| Umano    | Comune   |
| Elfo     | Elfico   |
| Gnomo    | Gnomesco |
| Mezzelfo | /        |
| Orco     | Orchesco |
| Mezzorco | /        |
| Nano     | Nanico   |
Maggioranza della popolazione parla comune, dato che è la lingua ufficiale della capitale [[Voxerrum]] e una gran parte delle razze ibride[^1] parlano le lingue di entrambe le loro metà.

[^1]: Mezzelfi e Mezzorchi.
